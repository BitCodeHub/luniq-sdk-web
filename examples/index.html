Luniq Web SDK demo

Open the browser console to see network calls. Replace the API key with your own to send real events.

// log output appears here

Code

// 1. install
npm install @luniq/sdk

// 2. initialize once at app start
import { Luniq } from "@luniq/sdk";
Luniq.start({ apiKey: "lq_live_...", endpoint: "https://uselunaai.com" });

// 3. send events
Luniq.identify("user_42", { plan: "pro" });
Luniq.track("button_click", { label: "checkout" });
